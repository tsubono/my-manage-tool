# store

store ディレクトリには [Vuex ストア](https://vuex.vuejs.org/) のファイルを入れます。Vuex ストアは Nuxt.js に付属していますが、デフォルトでは無効になっています。このディレクトリに index.js　ファイルを作成するとストアが有効になります。

[storeについて詳しく](https://ja.nuxtjs.org/guide/vuex-store)

***

- client.js
  - 取引先関連のstore
  
- label.js
  - ラベル関連のstore
  
- project.js
  - 案件関連のstore
  
- index.js
  - 未使用
  - Vuexの使用に必要
